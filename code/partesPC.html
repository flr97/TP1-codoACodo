Una Computadora es una máquina electrónica digital programable que ejecuta una serie de comandos para procesar los datos de entrada, obteniendo información que se envía a las unidades de salida.

Desde el punto de vista funcional, una computadora debe poseer al menos un CPU, una memoria y un dispositivo de entrada/salida, aunque existen otros componentes auxiliares que pueden ampliar su capacidad o asistir en conseguir una mejor experiencia para el usuario.
Además dependiendo de las especificaciones de la computadora, puede ser requerido distintos tipos de fuente de alimentación para garantizar una correcta alimentación energetica de los componentes
